# [voteforjulia.com](https://voteforjulia.com/)
![image](https://raw.githubusercontent.com/OwlbearMedia/voteforjulia/refs/heads/main/img/julia-rect-large.png)

The official campaign website for Julia Hamann, candidate for Mayor of Mankato.

## Tech Stack

- Frontend: Vue 3 + Vite + Vite SSG
- Styling: Tailwind CSS + custom CSS
- Backend API: Flask (Python)
- Frontend tests: Vitest + @vue/test-utils + jsdom
- CI/CD: GitHub Actions + SCP deploy

## Prerequisites

- Node.js 22+
- pnpm 11+
- Python 3.11+ (for API tests)

## Install Dependencies

Install frontend dependencies:

```bash
pnpm install
```

Install Python API dependencies:

```bash
python -m pip install -r api/requirements.txt
```

## Development

Start the local frontend dev server:

```bash
pnpm run dev
```

Preview a production build locally:

```bash
pnpm run build
pnpm run preview
```

## Build

Create the production static output in dist:

```bash
pnpm run build
```

Build details:

- Runs type-checking via vue-tsc.
- Generates static pages with Vite SSG.
- Produces deployable assets in dist.

## Testing

### Frontend Unit Tests (Vitest)

Run all frontend unit tests:

```bash
pnpm test
```

Run in watch mode:

```bash
pnpm run test:watch
```

Tests are organized in a dedicated directory:

- tests/unit/JuliaHeader.spec.ts
- tests/unit/JuliaFooter.spec.ts
- tests/unit/JuliaContactForm.spec.ts
- tests/unit/pages.spec.ts

### Python API Tests

Run backend API tests:

```bash
python -m unittest api/test_models.py api/test_email_service.py
```

## Deployment (GitHub Actions)

Deployment is defined in .github/workflows/deploy-scp.yml and runs on push to main.

Current CI/CD order:

1. Checkout repository
2. Setup pnpm + Node.js
3. Setup Python
4. Install Node dependencies
5. Install Python dependencies
6. Run frontend tests (pnpm test)
7. Run Python API tests (unittest)
8. Build site (pnpm run build)
9. Upload dist contents to server root (./public_html)
10. Upload api contents to server api directory (./api)

If either frontend or Python tests fail, deployment stops before build/upload.
